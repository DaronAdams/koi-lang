
# Koi-Lang

Dynamically typed scripting language based on python, rust, and javascript

[![koi-logos-transparent.png](https://i.postimg.cc/0QCbTDMq/koi-logos-transparent.png)](https://postimg.cc/CBzhDZsv)

## Features

- Classes
- Functions
- Error handling
- Global variables
- Conditionals
- Looping


## Syntax Examples 
```
// ------------ Printing
print "koi";
print true;
print "false";
print 20-100;

// ------------ Variables
var koi = "koi";
var a = 2;

// ------------ Functions
func add(a, b) {
    return a + b;
}
```
